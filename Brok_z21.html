Co znaczy skrót ID? oznacza unikalny identyfikator elementu w dokumencie HTML.
Jaki jest cel stosowania ID? Służy do jednoznacznego wskazania konkretnego elementu na stronie (np. do stylowania w CSS, manipulacji w JavaScript lub tworzenia odnośników).
Jakie jest znaczenie znaku # Znak # oznacza odwołanie do identyfikatora (ID) — używany np. w CSS (#menu) lub w linkach (#sekcja).
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? ID wywołujemy przez #nazwaID, np. w CSS: #header { ... } – oznacza to, że styl dotyczy elementu o atrybucie id="header".
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.
  1. ID musi być unikalne – nie może się powtarzać na stronie.
  2. Jeden element może mieć tylko jedno ID.
  3. ID ma wyższy priorytet w CSS niż klasy (class).
